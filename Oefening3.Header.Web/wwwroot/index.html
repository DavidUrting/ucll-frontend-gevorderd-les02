Een 15-seconden pagina...

Deze pagina blijft 15 seconden in de cache zitten. Na 15 seconden wordt deze terug opgehaald bij de server.

Een Cache-Control header wordt toegevoegd in Startup.cs.

Indien je test in Chrome, zie SO voor een waarschuwing. Best is dat je gewoon test in een nieuwe tab.

Je kan zien of de pagina uit de cache komt door naar de Network tab te kijken bij Developer tools: bij index.html zou (disk cache) moeten staan als je binnen de 15 seconden de pagina opvraagt (in een andere tab dus).
